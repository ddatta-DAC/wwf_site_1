WWF Data

WWF Data Details

Some details are as follows :

  • The data has 20000 rows.
  • The data is from 2007 to 2013.

The important attributes in data are as follows.

  1. Arrival Date
  2. Source Country ( eg. CN, JP )
  3. estimatedValue (dollar value)
  4. regionOfOrigin ( e.g. Far East, South America )
  5. portArrivalOrig
  6. portDepartureOrig
  7. coastalRegion ( which coast of US the shipment came in )
  8. Carrier ( e.g. Maersk )
  9. Consignee
  10. Consignee Address
  11. Harmonized Tariff Schedule (HS Code) [ HS Code is a 6 digit code for commodities by which countries levy tax on entry ]
  12. Weight
  13. Shipper
  14. Shipment Address
  15. Commodity_Desc [ Text information describing the item ]

Observation : Most of the trade is from China.
